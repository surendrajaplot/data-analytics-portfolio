# Hiring Process Analytics

## Project Overview

This project analyzes recruitment and workforce data to uncover patterns in hiring, compensation, departmental representation, and organizational structure. The objective was to generate actionable insights that can support strategic hiring decisions and improve workforce planning.

---

## Business Problem

Organizations require data-driven recruitment strategies to attract talent, maintain diversity, ensure equitable compensation, and optimize workforce allocation. This project investigates hiring data to identify trends and provide recommendations for improving the recruitment process.

---

## Objectives

- Analyze gender distribution among new hires.
- Evaluate average salary offerings.
- Examine salary distribution patterns.
- Assess departmental workforce representation.
- Investigate position tier distribution.
- Identify data quality issues including missing values and outliers.

---

## Methodology

### Data Preparation

- Missing value detection and handling.
- Data validation and cleaning.
- Outlier detection using statistical methods.
- Data transformation and categorization.

### Data Analysis

- Gender distribution analysis.
- Salary trend analysis.
- Salary frequency distribution.
- Department-wise workforce analysis.
- Position tier distribution analysis.

### Visualization

- Bar Charts
- Pie Charts
- Salary Distribution Histograms
- Summary Tables
- Dashboard Components

---

## Key Analyses

### Gender Distribution Analysis

Examined workforce diversity by calculating the proportion of male and female hires.

### Salary Analysis

Calculated average compensation offered to employees and evaluated overall compensation trends.

### Salary Distribution

Created salary bands and analyzed the frequency of employees within each salary range.

### Departmental Analysis

Investigated workforce allocation across departments to identify staffing patterns.

### Position Tier Analysis

Evaluated the distribution of employees across organizational levels and job tiers.

---

## Key Findings

- Gender representation trends highlighted workforce diversity patterns.
- Salary analysis provided insights into organizational compensation strategies.
- Departmental distribution revealed workforce concentration across business functions.
- Position tier analysis supported workforce planning and talent development initiatives.
- Outlier detection improved data quality and analytical reliability.

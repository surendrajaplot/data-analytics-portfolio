# Key Excel Formulas Used

## Gender Analysis

```excel
=COUNTIF(gender_range,"Male")
=COUNTIF(gender_range,"Female")
```

## Average Salary

```excel
=AVERAGE(salary_range)
```

## Salary Distribution

```excel
=FREQUENCY(salary_range,bins_range)
```

## Department Analysis

```excel
=COUNTIF(department_range,"Department Name")
```

## Position Tier Analysis

```excel
=COUNTIF(position_tier_range,"Tier 1")
```
